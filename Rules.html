Rules

The wikipedia game doesn't have any rules that are set in stone, however there are some optional
rules.

Optional Rules

CTRL + F

CTRL + F is generally unallowed in the wikipedia game, especially if you are
racing against somebody. If you would like to use this shortcut, make sure
the person you are racing against knows that they are allowed to use it aswell.

Previous Page

People also may or may not be able to go back to a previous page, and often going back to the
previous page can count as a click.

For more information visit the Official Site
